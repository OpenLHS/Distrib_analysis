# logistic_regression_penalized
Distributed analytics with vertically partitioned data under the penalized logistic regression model.

## Repository structure

1. Methods 
The material pertaining to each method is located in its related folder, including generic code, requirements and examples.
Each method folder is self-contained.

## Methods 

- RidgeLog-V: One-shot and lossless ridge logistic regression

- RidgeLog-V2: One-shot and lossless ridge logistic regression that does not require sharing the response vector and allows cross-validation for the choice of penalty
